A modern, high-performance portfolio website built with **React**, **Three.js**, and **GSAP**. This portfolio features a unique interactive 3D Lanyard, seamless multi-language support (ID/EN), and premium animations for a state-of-the-art user experience.

## ✨ Key Features

-   **🪪 Interactive 3D Lanyard**: A physics-based 3D identification card that reacts to mouse movement and gravity (built with React Three Fiber & Rapier).
-   **🌐 Multi-language (ID/EN)**: Instant language switching across the entire site without page refreshes.
-   **⏳ Premium Preloader**: A custom percentage-based loader with a smooth "shutter reveal" exit animation.
-   **🎭 Split Text Animations**: Dynamic heading animations using GSAP and ScrollTrigger.
-   **📱 Fully Responsive**: Optimized for all screen sizes, from mobile phones to high-resolution desktops.
-   **🎨 Dark Tech Aesthetic**: Curated color palette featuring glassmorphism and modern typography.

## 🚀 Tech Stack

-   **Framework**: [React.js](https://reactjs.org/)
-   **Build Tool**: [Vite](https://vitejs.dev/)
-   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
-   **3D Engine**: [Three.js](https://threejs.org/) / [@react-three/fiber](https://github.com/pmndrs/react-three-fiber)
-   **Physics**: [@react-three/rapier](https://github.com/pmndrs/react-three-rapier)
-   **Animation**: [GSAP](https://greensock.com/gsap/) / [@gsap/react](https://github.com/greensock/react)

## 🛠️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/dutautes/portofolio-4.git
    cd portofolio-duta
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```

## 📂 Project Structure

-   `src/components/`: Modular UI components (Hero, Navbar, Lanyard, etc.).
-   `src/context/`: State management for language preferences.
-   `src/translations.js`: Centralized dictionary for multi-language content.
-   `src/assets/`: 3D models (`card.glb`) and static images.
